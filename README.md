Patient Data Analysis Project

This project analyzes a healthcare dataset using Excel, SQL, and Power BI.

Contents
- patients.csv — Raw dataset
- patients_analysis.sql — SQL queries for analysis
- patients_dashboard.pbix — Power BI dashboard
- patients_report.pdf — PDF export of dashboard

Highlights
- Gender and condition breakdown
- Billing patterns
- Average age, admission trends
- Visual dashboards in Power BI
